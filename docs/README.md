# docs

<!-- okq:begin -->
## Knowledge base

This directory is an [Open Knowledge Format](https://github.com/GoogleCloudPlatform/knowledge-catalog/tree/main/okf)
(OKF) bundle. Explore it with [okq](https://github.com/mikevalstar/okq):

    okq search "<topic>"     # ranked full-text
    okq find --type adr      # filter by frontmatter
    okq stats                # overview
    okq new adr "<title>"    # add a decision
<!-- okq:end -->
